# Agency AI

[![Live Demo](https://img.shields.io/badge/Live%20Demo-vercel.app-black?style=flat&logo=vercel)](https://agency-ai-nu-blush.vercel.app/)
[![GitHub stars](https://img.shields.io/github/stars/eyob42/agency-ai)](https://github.com/eyob42/agency-ai/stargazers)
[![GitHub issues](https://img.shields.io/github/issues/eyob42/agency-ai)](https://github.com/eyob42/agency-ai/issues)
[![MIT License](https://img.shields.io/github/license/eyob42/agency-ai)](https://opensource.org/licenses/MIT)

A modern, responsive marketing agency website built with React and Tailwind CSS. Features smooth animations, a custom cursor, dark mode, and a fully responsive design.

## ✨ Features

- 🎨 **Modern Design** - Clean, professional layout perfect for creative agencies
- 🌓 **Dark Mode** - Seamless light/dark theme switching with persistent localStorage
- 🖱️ **Custom Cursor** - Smooth animated cursor with hover effects
- 📱 **Fully Responsive** - Optimized for all devices (mobile, tablet, desktop)
- 🚀 **Smooth Animations** - Subtle transitions and interactive elements
- 📧 **Contact Form** - Integrated with Resend email service
- 🔔 **Toast Notifications** - User-friendly feedback system
- ⚡ **Optimized Performance** - Built with Vite for fast loading

## 🛠️ Tech Stack

| Technology           | Description               |
| -------------------- | ------------------------- |
| React 19             | Frontend framework        |
| Tailwind CSS 4       | Styling and utilities     |
| Vite                 | Build tool and dev server |
| Framer Motion        | Animations                |
| Lucide React         | Icons                     |
| React Hook Form      | Form handling             |
| React Hot Toast      | Notifications             |
| React Email + Resend | Email service             |
| ESLint + Prettier    | Code quality              |

## ⚙️ Installation

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

```sh
# Clone the repository
git clone https://github.com/eyob42/agency-ai my-project

# Navigate to the project directory
cd my-project

# Install dependencies
npm install

# Run the application
npm start
```

> 📝 **Note:** Ensure that **Node.js v18+** is installed before running this project.
